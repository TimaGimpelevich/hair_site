# Парикмахерская СЕТЬ

Отдельный статический проект для базовой структуры сайта салона.

## Как открыть

Из папки `set-hair-site`:

```bash
python3 -m http.server 4174
```

Потом открыть `http://127.0.0.1:4174`.

## Что уже есть

- `index.html` — главная страница
- `styles.css` — стили сайта
- `content.js` — редактируемый контент: услуги, работы, новости и ссылка записи
- `script.js` — рендер карточек, мобильное меню, подстановка ссылок записи
- `assets/images/logo-source.jpg` — логотип клиента
- `assets/videos/hero.mp4` — видео для первого экрана

## Что менять дальше

- ссылка онлайн-записи: поле `bookingUrl` в `content.js`
- видео в первом экране: поле `heroVideoUrl` в `content.js`
- новости: массив `news` в `content.js`
- карточки работ и подписи к фото: массив `works` в `content.js`
